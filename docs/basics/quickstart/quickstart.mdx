# Get started with Fused!

Use the [Fused Python SDK](/python-sdk) to run a workflow in your favorite IDE, Jupyter
Notebook, or Python environment.

Understand Fused in 5 minutes by playing!

Open a new Python environment and follow these *very short* steps.

Install the Fused Python SDK.

```python
!pip install fused -q
```

Paste and run this snippet. This snippet imports then run a UDF from the
[UDF catalog GitHub repo](https://github.com/fusedio/udfs/tree/main).

```python
import fused

udf = fused.load("https://github.com/fusedio/udfs/tree/main/public/DuckDB_NYC_Example")
gdf = udf.run_local()
gdf
```

# Quickstart

## Fast Track ⏱️

Understand Fused in 5 minutes by playing!

Open a new Python environment and follow these *very short* steps.

Install the Fused Python SDK.

```bash
pip install fused
```
Paste and run this snippet. This snippet shows how to import then run a UDF from the [UDF catalog GitHub repo](https://github.com/fusedio/udfs/tree/main).

```python
import fused

udf = fused.load("https://github.com/fusedio/udfs/tree/main/public/DuckDB_NYC_Example")
gdf = udf.run_local()
gdf
```
